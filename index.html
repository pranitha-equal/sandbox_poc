LowCardinality columnar encoding

ClickHouse uses LowCardinality(String) for attribute_name, sandbox_id, and pdf_status — repeated string values are dictionary-encoded and block-compressed together, giving near-zero marginal cost per repeated attribute name.   Crossover point: CH wins on 9/10 queries beyond ~20% scale (~5,000 PG rows). Below that threshold, PG's lower overhead wins on small scans.

Q1 — Run Consolidation: Aggregate avg end-of-day balance, total debit, and total credit across all accounts in a single run. Tests wide-scan aggregation on all rows sharing a run_id.

Q2 — Selected Account Comparison: Fetch and compare full key metrics for a fixed set of accounts by pdf_id. Tests point-lookup + narrow aggregation on specific IDs.

Q3 — Top Accounts By Balance: Rank all accounts in the dataset by avg end-of-day balance and return the top 10. Tests a full-table sort with ORDER BY + LIMIT.

Q4 — Global Account Averages: Compute AVG of balance, debit amount, and credit amount across every account in the full dataset. Tests multi-attribute full-table aggregation.

Q5 — Account Drilldown: Read all stored attributes for one specific account by pdf_id. Pure single-row point lookup — PG's primary strength due to heap + index access pattern.

Q6 — Run vs Run Comparison: Side-by-side aggregate comparison of two specific runs — avg balance, total debit, total credit for each. Tests filtering on two distinct run_ids simultaneously.

Q7 — Run Summary Dashboard: Per-run multi-metric summary: avg balance, total debit, total credit, and salary flag count — all in one pass, grouped by run_id. Tests multi-attribute pivot aggregation.

Q8 — Credit Active Ranking: Rank accounts by total credit amount, filtered to those with at least one credit transaction. Tests filtered aggregation + top-N ordering on a derived metric.

Q9 — Attribute Completeness: Count how many accounts have each attribute populated. Tests a GROUP BY attribute_name with COUNT DISTINCT pdf_id — a data-coverage audit across the full dataset.

Q10 — Multi Metric Leaderboard: Top 10 accounts ranked by a composite score across avg balance, total debit, and total credit simultaneously. Tests multi-attribute pivot + composite ordering in a single query.
